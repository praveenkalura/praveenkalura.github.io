# praveenkalura

Personal website of Dr. Praveen Kalura.

This repository is arranged as a static GitHub Pages site. The page shell lives at `index.html`, and the compiled React app is served from `static/js/bundle.js`.

## Local preview

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000`.

## GitHub Pages

Publish from the repository root on the default branch. The `.nojekyll` file keeps GitHub Pages from running Jekyll over the static bundle.
